Q.7): Write a function that accepts parameter n and returns factorial of n.

Ans: 
             
                function factorial(n){
                    if(n == 0 || n == 1){
                        return 1;
                    
                    }else{
                        return n * factorial(n-1);
                    }
                }
                let n = 5;
                answer = factorial(n)
                console.log("The factorial of " + n + " is " + answer);
                
    

    Output: The factorial of 5 is 120
